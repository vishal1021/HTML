List

- There are three types of lists
1) Ordered list
2) Unordered list
3) Defination list

Example -

Ordered List -

Default

  1. Html
  2. Css
  3. Javascript

For start with customized numbers use start="number" attribute.

  1. Html
  2. Css
  3. Javascript

For start with alphabates use type="Alphabate" attribute.

  1. Html
  2. Css
  3. Javascript

For start with roman numbers use type="i" attribute.

  1. Html
  2. Css
  3. Javascript

For start with capital roman numbers use type="I" attribute.

  1. Html
  2. Css
  3. Javascript

Unordered List -

Default

  • Lion
  • Tiger
  • Leopard

For start with circle use type="circle" attribute.

  • Lion
  • Tiger
  • Leopard

For start with circle use type="square" attribute.

  • Lion
  • Tiger
  • Leopard

Note - There are only three values of type attribute which are disk(default),circle and square.

Defination List -

Default

Html
It is client side scripting language.
Css
Cascading style sheet is used to formatting html pages.
